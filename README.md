# Binary Classifier from Scratch (NumPy Only)

## Introduction

This project is a binary classification model built completely from scratch using only NumPy.  
No machine learning libraries are used, just math and the `numpy` library.

## Goal

The goal is to understand how a simple classifier works at a fundamental level:

- How predictions are computed  
- How loss is calculated  
- How gradients are derived  
- How weights are updated through gradient descent  

This is about understanding the mechanics behind logistic regression.
